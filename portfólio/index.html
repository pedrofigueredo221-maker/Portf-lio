Media Queries

Exemplo de responsividade usando HTML e CSS.

O que é?

Media Queries são recursos do CSS utilizados para adaptar o layout de um site em diferentes tamanhos de tela.

Como funciona?

O CSS identifica o tamanho da tela do dispositivo e aplica estilos diferentes quando necessário.

Exemplo Prático

Redimensione a janela do navegador para visualizar a mudança de layout.
